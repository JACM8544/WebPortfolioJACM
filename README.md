# Web Portfolio — Jorge Armando Contreras Mercado

Personal web portfolio showcasing professional experience, technical skills, and key projects.

## Live site

Deployed on **GitHub Pages**.

## Features

- **Single-page responsive design** — dark theme with black & gold palette
- **EN/ES language toggle** — full bilingual support for all content
- **Highlights carousel** — auto-advancing with fade transitions and dot navigation
- **Animated timeline** — vertical line with glowing pulse on current position
- **Animated stat counters** — count-up animation triggered on scroll
- **Scroll fade-in** — sections and staggered children animate into view
- **Hover effects** — lift on chips and project card
- **SVG icons** — inline icons for each skill category
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript
